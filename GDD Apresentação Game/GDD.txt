Game Design Document

Título
O nome do nosso jogo se chama Destino de Sonia.

Descrição do Jogo
O nosso jogo é minigame composto de uma cena curta, onde a brava heroína Sonya adentra uma fortaleza com o objetivo de capturar um grande tesouro. Contudo, essa fortaleza não é um local qualquer, ela é habitada pelo temido Lord Cruel, um mago malévolo que patrulha a redondeza com suas terríveis criaturas conjuradas.

Tipo de Jogo / Gênero
Destino de Sonia é um jogo em formato 3D de estilo RPG, que é para ser jogado offline e se compõe de missões curtas para conclusão. Nesta versão, apenas uma missão é oferecida para o usuário final.
